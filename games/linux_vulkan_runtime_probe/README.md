# Linux Vulkan Runtime Probe

`linux_vulkan_runtime_probe` is a Linux-only package validation executable for the Environment Highest Commercial Readiness v1 Linux Vulkan row.

It loads the packaged weather-solver SPIR-V artifact, runs the first-party Linux runtime host offscreen Vulkan compute/readback probe, emits package-visible readiness counters, and keeps Windows Vulkan, Android Vulkan, all-platform readiness, commercial readiness, and broad `environment_ready` unclaimed.
